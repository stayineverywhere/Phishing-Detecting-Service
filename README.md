## 7. 탐지 알고리즘 (Risk Scoring Logic)

본 시스템은 딥러닝(KoBERT) 점수와 규칙 기반 점수를 결합한 하이브리드 판정 방식을 사용합니다.

- **최종 판정 공식**: `z = -5 + 5*K + 5*P + 4*(AI^2) + 8*U + 8*C`
  - **K (Keywords)**: 위험 키워드 포함 여부 (일상어 제외)
  - **P (Patterns)**: 피싱 문장 패턴 유사도 - 정상 문장 패턴 유사도 (TF-IDF 기반, 0 이하 컷)
  - **AI (Deep Learning)**: KoBERT 모델 점수 (정확도를 위해 제곱값 사용)
  - **U (Urgency)**: 긴급 상황 유도 및 압박 수위
  - **C (Context)**: 금전 요구 및 기관 사칭 맥락
- **특징**: "엄마 나 도착했어"와 같은 일상 안부 인사의 오탐지를 최소화하고, 실제 금전 요구가 포함된 피싱은 강력하게 탐지하도록 최적화되었습니다.

## 8. 실시간 탐지 및 자동 연동 기능

본 시스템은 사용자의 개입을 최소화하기 위한 실시간 자동화 기능을 포함하고 있습니다.

### 실시간 문자(SMS) 탐지
- **자동 감지**: 시스템 브로드캐스트 리시버(`SmsReceiver`)를 통해 수신된 모든 문자를 실시간 모니터링합니다.
- **자동 분석**: 문자가 수신되면 앱이 즉시 실행되며, 내용을 자동으로 입력창에 채운 뒤 별도의 클릭 없이 **즉시 피싱 분석을 수행**합니다.

### 실시간 통화 음성 인식(STT) 탐지
- **상태 감지**: 전화 발신/수신 상태를 실시간으로 감지(`CallReceiver`)하여 감시 서비스를 시작합니다.
- **실시간 STT**: 통화 중 목소리를 백그라운드 서비스(`CallService`)에서 텍스트로 즉시 변환합니다.
- **실시간 업데이트**: 변환된 텍스트는 분석 화면에 실시간으로 입력되어 통화 중에도 위험도를 즉각 파악할 수 있습니다.

### 시연 및 테스트 방법
- **문자 테스트**: 안드로이드 에뮬레이터의 `Extended Controls > Phone` 메뉴에서 가짜 SMS를 전송하여 자동 분석 동작을 확인합니다.
- **전화 테스트**: 통화 연결 후 앱의 STT 분석 화면에서 실시간으로 텍스트가 채워지는지 확인합니다. (스피커폰 사용 권장)
